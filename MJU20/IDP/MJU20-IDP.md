# MJU20-IDP

[Up](../MJU20.md) [Here](.)

**Förutsättningar:** covid-19 hade börjat
sprida sig och skolsals­undervis­ningen ersattes
av undervisning på zoom. Start 17 mars 2020.
Det tog ett tag innan vi fick ordning på
undervisningen, men när MJU20 började, var den
konsoliderad (och vi körde distansutbildningar
fram till 2025). Programmet MJU20 startade på
höst­terminen 2020. Första eller andra kursen
var **Introduktion till datorer och
programmering**. Denna innehöll:

- operativsystem inklusive kommandorad och
  filsystem,
- annat för att orientera sig i systemet, bl.a.
  PATH
- introduktion till programmering (kodande) i
  C#,
- med mera
